How this works

      
    
        
  • Pick a base color (color picker or hex input), or upload a
          reference image — Palette Forge reads its actual pixel data
          and finds a dominant color via HSL-bucket vibrancy weighting
          (real image analysis, run entirely in your browser).
    • 
        
  • Choose a harmony type. Each one is real color-wheel math in HSL
          space: complementary (180° hue rotation), analogous
          (±30°), triadic (120°/240°), tetradic
          (90°/180°/270°), monochromatic (fixed hue/saturation,
          varying lightness).
    • 
        
  • Every swatch shows its real WCAG contrast ratio against white
          and black text, using the actual WCAG 2.x relative-luminance
          formula — labeled AA/AAA pass or fail, not approximated.
    • 
        
  • Export as CSS custom properties (free), or unlock Tailwind
          config + Style Dictionary JSON.
    • 
        
  • Nothing you enter or upload ever leaves your browser. No
          accounts, no cloud save, no history — this is a one-shot tool by
          design.
